# Prodigy Whitelist Portal

A secure, Discord-authenticated whitelist management system for FiveM.

## Features
- Discord OAuth2 Authentication.
- Randomized Quiz Mechanics with cooldowns.
- Professional Green-Accent UI (Prodigy Theme).
- Automated Role Management via Discord Bot.

## Setup
1. Clone the repo.
2. Run `npm install`.
3. Create a `.env` file with your credentials (`DISCORD_TOKEN`, `CLIENT_ID`, etc.).
4. Run `npm run dev` to start locally.
